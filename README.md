# 親親寶貝 圖鑑 (KissKiss Baby DB)

一個深色遊戲風的角色圖鑑網站，可以收集親親幫的各路神獸。

## 檔案結構

```
KissKissBabyDB/
├── index.html         ← 主網頁 (雙擊打開就能看)
├── characters.js      ← 角色資料檔 (手動新增永久角色請編輯這裡)
├── images/
│   └── 001.png        ← 親親獸的圖片 (請把你上傳的那張圖存到這裡)
└── README.md
```

## 快速開始

1. 把你上傳的「親親獸」那張粉紅色圖片存成 `images/001.png`
2. 用瀏覽器雙擊打開 `index.html` 即可使用

> 有些瀏覽器（特別是 Chrome）為了安全性，直接用 `file://` 打開時可能會擋本機圖片。
> 如果遇到圖片無法顯示，可以在資料夾裡跑：
> - Python：`python -m http.server 8080` → 打開 <http://localhost:8080>
> - 或者用 VS Code 的 Live Server 擴充

## 功能

- **列表 + 詳細頁**：點擊任一卡片可放大看完整資料
- **搜尋 / 篩選**：支援名稱、來源、興趣關鍵字搜尋
- **響應式設計**：手機、平板、電腦都能看
- **頻道連結可點**：點一下就開新分頁
- **網頁投稿**：右上角「＋ 投稿新增」可當場新增角色 (含圖片上傳)
- **匯出 JSON**：把所有角色 (含投稿) 一鍵下載為 `characters.json`

## 新增角色的兩種方式

### 方式 A：網頁投稿 (暫存在瀏覽器)

點右上角「＋ 投稿新增」→ 填寫資料 → 上傳圖片 → 送出。
這些資料會存在**你的瀏覽器 localStorage**裡，換瀏覽器/清 cookie 就會消失。

### 方式 B：永久新增 (推薦長期用)

1. 點右上角「📤 匯出」下載完整的 `characters.json`
2. 打開下載的 JSON，把內容搬到 `characters.js` 的 `CHARACTERS` 陣列裡
3. 把對應圖片放到 `images/` 資料夾
4. 重新整理網頁，新角色就永久存在了

或者直接手動編輯 `characters.js`：

```js
const CHARACTERS = [
  {
    id: "001",
    name: "親親獸",
    source: "蕭發仔 (親親幫幫主)",
    channel: "https://www.twitch.tv/m989876525",
    interests: ["收護貝費", "剪紙", "當太監"],
    image: "images/001.png",
    desc: "傳說中粉紅色的親親獸..."
  },
  // 在這裡新增下一隻
  {
    id: "002",
    name: "???",
    ...
  }
];
```
